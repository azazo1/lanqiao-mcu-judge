# key seg

按下矩阵键盘的按键 S4-S19 (没有 S0-S3, 板子设计如此), seg 和 led 会有对应的变化.

初始上电的时候, 当没有按键按下 seg 显示 0.

当按键 S4-S11 按下的时候, L1-L8 各自在按顺序对应的按键按下的时候切换亮灭.
S12-S19 按顺序和各自对应的 S4-S11 在 led 中的效果相同, 都是按下的时候让对应的 led 切换亮灭.

比如 S5 按下瞬间, L2 变亮, 或者 L2 变暗.

seg 显示的是当前按键位码 `或` 结果.
比如按下 S4, 显示 1,
按下 S14, 显示 1024,
按下 S14+S19, 显示 33792.
