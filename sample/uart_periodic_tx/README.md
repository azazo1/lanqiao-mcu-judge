# 串口分段

本 sample 用来验证 UART1 的长时间累计读取行为.

程序上电后使用 UART1 周期性发送报文, 发送节奏如下:

- 第 1 条在约 120ms 发出, 内容为 `OK`
- 之后每隔约 280ms 再发 1 条
- 内容按 `OK`, `ERROR`, `OK`, `ERROR` 交替

数码管左两位显示已经发送完成的报文数量, 其余位保持空白.

对应 judge 会验证两种读取方式:

- `run_ms(1000)` 之后直接 `uart_take()`, 结果会把 4 条报文合并成 1 个字符串
- `run_ms(1000)` 之后使用 `uart_take(10)`, 可以按 10ms 空闲间隔逐段读出 `OK`, `ERROR`, `OK`, `ERROR`
