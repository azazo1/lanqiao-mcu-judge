# NE555

这个样例专门用于验证 `NET_SIG -> SIG_OUT -> P3.4/T0` 这条测频链.

程序行为很简单:
- 左侧 3 位显示当前按键编号.
- 右侧 5 位显示最近一个测频窗口内统计到的频率值.
- `Timer0` 工作在外部计数模式, 用来统计来自 `P3.4/T0` 的 NE555 输入脉冲.

评测时推荐:
- 先保持默认跳帽状态, 此时即使调用 `set_frequency_hz(...)`, 单片机也读不到 NE555 输出.
- 再显式调用 `jumper_on(NET_SIG, SIG_OUT)`, 把 NE555 输出接到 `P3.4/T0`.
- 最后设置不同的频率值, 检查数码管右侧 5 位是否落在允许范围内.
